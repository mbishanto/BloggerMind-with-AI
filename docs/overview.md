# Overview

## What Is Blogger Mind?

Blogger Mind is a community blogging platform where users can publish posts, interact with content through likes, and access AI-assisted writing tools. It is built with PHP and MySQL and deployed on standard cPanel shared hosting.

**Live site:** [https://bloggersminds.com/](https://bloggersminds.com/)

## What It Does

- Users can register, verify their email, log in with OTP verification, and manage their profiles.
- Users can create, edit, and publish posts using the TinyMCE rich text editor. Posts can be categorized and assigned a featured image.
- The homepage displays a community feed with infinite scroll pagination, category filtering, and sidebar widgets (top authors, trending posts, most liked content, latest news headlines).
- An AI Assistant provides chat-based help across four modes: general chat, SEO writing, code fixing, and tool guidance. It uses multiple AI providers (Groq, OpenRouter, Gemini) with automatic fallback.
- A collection of client-side utility tools (password generator, duplicate line remover, calculators, image tools, text tools) are available without server round-trips.

## Target Audience

- Bloggers and content creators looking for a community publishing platform.
- Developers interested in examining a PHP/MySQL codebase with real AI integration.
- Anyone exploring AI-assisted software development as a development methodology.

## What It Is Not

- **Not a SaaS product.** The source code is available for self-hosting.
- **Not a WordPress alternative.** It is a from-scratch build with its own architecture.
- **Not an AI content farm.** AI assists with writing suggestions and research, but all content is created and published by human users.
